### **Что такое сетевой адаптер?**
**Сетевой адаптер (Network Interface Car, NIC)** - это аппаратное устройство, которое позволяет компьютеру подключаться к сети и обмениваться данными.

**Другие названия**: сетевая карта, Ethernet-адаптер, LAN-адаптер.

##### **Основные задачи адаптера**
- **Преобразование данных** - из параллельного формата ПК в последовательный для передачи по сети
- **Формирование кадров** - добавление MAC-адресов, контрольных сумм
- **Управление доступом к среде** - реализация протоколов CSMA/CD (Ethernet)
- **Буферизация данных** - временное хранения кадров при приёме/передачи 

### Функции сетевого адаптера
#### Физическая адресация
Каждый адаптер имеет уникальный MAC-адрес (48 бит), прошитый производителем. Используется для идентификации устройства в локальной сети.

#### Передача/приём данных
Преобразует биты в электрические/оптические сигналы (передача) и обратно (приём). Контролирует целостность с помощью CRC.

#### Управление потоком
Регулирует скорость передачи для предотвращения переполнения буфера получателя (Flow Control).

### Характеристики сетевых адаптеров
#### Скорость передачи
10 Мбит/с — Ethernet (устаревший)
100 Мбит/с — Fast Ethernet
1 Гбит/с — Gigabit Ethernet (стандарт)
10 Гбит/с, 25/40/100 Гбит/с  — для серверов и ЦОД

#### Другие параметры
Тип интерфейса — PCI, PCIe, USB, M.2
Дуплекс — полудуплекс / полный дуплекс
Поддержка Wake-on-LAN (WoL) — удалённое включение ПК
Поддержка VLAN, QoS — для корпоративных сетей
