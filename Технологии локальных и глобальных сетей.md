## Технология Ethernet

**Ethernet** — самая распространенная технология проводных ЛВС. Стандарт IEEE 8023 Работает на канальном и физическом уровнях OSI

### Основные характеристики

**Метод доступа:** CSMA/CD (обнаружение коллизий)

**Топология:** Логическая — шина, физическая — звезда (через коммутатор)

**Кабель:** Витая пара (UTP/STP), оптоволокно

### Стандарты Ethernet

|Стандарт|Скорость|Среда|Дальность|
|---|---|---|---|
|10Base-T|10 Мбит/с|UTP Cat3|100 м|
|100Base-TX|100 Мбит/с|UTP Cat5|100 м|
|1000Base-T|1 Гбит/с|UTP Cat5e|100 м|
|10GBASE-T|10 Гбит/с|UTP Cat6a|100 м|
|100GBASE-SR1|100 Гбит/с|Optika MMF|100 м|
## Структура кадра Ethernet

**Кадр Ethernet (IEEE 802.3)** — формат данных канального уровня для передачи по сети

|Preamble|SFD|MAC назначения|MAC источника|Type/Length|Данные (Payload)|FCS|
|---|---|---|---|---|---|---|
|7 байт|1 байт|6 байт|6 байт|2 байт|46-1500 байт|4 байт|

### Описание полей

- **Preamble + SFD** — Синхронизация и начало кадра
    
- **MAC-адреса** — Источник и назначение (48 бит каждый)
    
- **Type/Length** — Тип протокола (IPv4, IPv6) или длина данных
    
- **Payload** — Полезные данные (46–1500 байт, MTU = 1500)
    
- **FCS (Frame Check Sequence)** — Контрольная сумма CRC-32 для обнаружения ошибок
## Технология Token Ring

**Token Ring (IEEE 802.5)** — технология ЛВС с кольцевой топологией и передачей маркера (token) Разработана IBM в 1980-х

### Принцип работы

1. Свободный маркер (token) циркулирует по кольцу
    
2. Станция захватывает маркер для передачи
    
3. Данные передаются в кадре по кольцу
    
4. Получатель копирует данные и помечает кадр
    
5. Отправитель удаляет кадр из кольца
    
6. Выпускается новый свободный маркер
    

### Характеристики

**Скорость:** 4 или 16 Мбит/с

**Метод доступа:** Передача маркера (детерминированный)

**Топология:** Физическая звезда (MAU), логическое кольцо

**Преимущество:** Нет коллизий, гарантированный доступ

### Современное состояние

Token Ring устарела и практически не используется Вытеснена Ethernet из-за более высокой скорости, низкой стоимости и простоты Последние продукты сняты с производства в начале 2000-х
## Технология FDDI

**FDDI (Fiber Distributed Data Interface)** — технология ЛВС на оптоволокне с кольцевой топологией Разработана ANSI в 1980-х для магистральных сетей

### Основные характеристики

**Скорость:** 100 Мбит/с

**Среда:** Оптоволокно (многомодовое/одномодовое)

**Дальность:** до 200 км (одномодовое)

**Топология:** Двойное кольцо (dual ring)

**Метод доступа:** Передача маркера

### Отказоустойчивость

**Двойное кольцо:**

- Первичное кольцо — основная передача данных
    
- Вторичное кольцо — резервное (обратное направление)
    

**При обрыве одного кольца:** Станции автоматически переключаются на вторичное кольцо, образуя единое кольцо Сеть продолжает работать

**Применение:** Магистрали, критичные инфраструктуры

> **Современное состояние:** FDDI устарела Заменена Gigabit Ethernet и 10G Ethernet на оптике, которые дешевле и быстрее Используется только в старых инфраструктурах

## Беспроводные локальные сети (Wi-Fi)

**Wi-Fi (Wireless Fidelity, IEEE 802.11)** — семейство стандартов беспроводных ЛВС Работает на частотах 2.4 ГГц, 5 ГГц, 6 ГГц (Wi-Fi 6E)

### Метод доступа: CSMA/CA

**CSMA/CA (Collision Avoidance)** Предотвращение коллизий через:

- Слушание среды перед передачей
    
- Случайный отсчёт (backoff)
    
- RTS/CTS (запрос/разрешение)
    
- ACK (подтверждение получения)

|Стандарт|Частота|Макс скорость|Год|
|---|---|---|---|
|80211b|2.4 ГГц|11 Мбит/с|1999|
|80211g|2.4 ГГц|54 Мбит/с|2003|
|80211n (Wi-Fi 4)|2.4/5 ГГц|600 Мбит/с|2009|
|80211ac (Wi-Fi 5)|5 ГГц|6.9 Гбит/с|2014|
|80211ax (Wi-Fi 6)|2.4/5/6 ГГц|9.6 Гбит/с|2019|

**Безопасность Wi-Fi:** WEP (устаревший, взломан) → WPA → WPA2 → **WPA3 (современный стандарт, шифрование AES-256)**
## Технологии глобальных сетей (WAN)

**WAN (Wide Area Network)** — глобальные сети, соединяющие удалённые узлы через телекоммуникационные каналы (телефонные линии, спутники, оптика)

### Выделенные линии (Leased Lines)

Постоянное соединение точка-точка Высокая стоимость, гарантированная полоса Примеры: T1 (1.5 Мбит/с), E1 (2 Мбит/с)

### Frame Relay

Коммутация пакетов через виртуальные каналы (PVC) Устаревшая технология, заменена MPLS

### ATM (Asynchronous Transfer Mode)

Коммутация ячеек фиксированного размера (53 байта) Использовалась в магистралях Вытеснена IP/MPLS

### MPLS (Multiprotocol Label Switching)

Современная технология коммутации по меткам Высокая производительность, QoS, traffic engineering

### xDSL (ADSL, VDSL)

Широкополосный доступ по телефонным линиям ADSL — до 24 Мбит/с, VDSL2 — до 100 Мбит/с

### PON (Passive Optical Network)

FTTH — оптика до дома GPON — до 2.5 Гбит/с, XG-PON — до 10 Гбит/с Современный стандарт
## Принципы межсетевого взаимодействия

**Межсетевое взаимодействие (Internetworking)** — объединение нескольких сетей с разными технологиями в единую инфраструктуру через маршрутизаторы

### Основные принципы

- **Маршрутизация (Routing)** — Выбор оптимального пути от источника к назначению через несколько сетей Протоколы: RIP, OSPF, BGP
    
- **Адресация** — Глобально уникальные IP-адреса обеспечивают идентификацию узлов в любой сети мира
    
- **Инкапсуляция** — Данные одного протокола упаковываются в кадры другого Пример: IP-пакет → Ethernet-кадр
    
- **Фрагментация** — Разбиение больших пакетов на меньшие для прохождения через сети с разным MTU
    
- **Протокол IP** — Единый сетевой протокол, работающий поверх любых канальных технологий (Ethernet, Wi-Fi, PPP)
## Роль маршрутизатора в межсетевом взаимодействии

**Маршрутизатор (Router)** — устройство сетевого уровня (L3), соединяющее несколько сетей и пересылающие пакеты между ними на основе IP-адресов

### Функции маршрутизатора

- Маршрутизация пакетов — выбор пути по таблице маршрутизации
    
- Объединение сетей — связывает LAN, WAN, интернет
    
- Фильтрация трафика — ACL (Access Control Lists), firewall
    
- NAT/PAT — трансляция адресов (частные ↔ публичные)
    
- Broadcast-изоляция — не пропускает broadcast между сетями
### Протоколы маршрутизации

**Внутренние (IGP):**

- RIP (Routing Information Protocol) — дистанционно-векторный, до 15 хопов
    
- OSPF (Open Shortest Path First) — состояния каналов, быстрая сходимость

**Внешние (EGP):**

- BGP (Border Gateway Protocol) — протокол интернета, маршрутизация между AS
