#### **1. Сетевая модель**

**Определение:** Абстрактная многоуровневая структура, описывающая передачу данных между устройствами. Каждый уровень выполняет строго определённые задачи и взаимодействует только со смежными уровнями

**Два основных подхода:**

1. **Эталонная модель OSI** (7 уровней, разработана ISO)

2. **Модель TCP/IP** (4 уровня, основа Интернета)

#### **2. МОДЕЛЬ OSI (OPEN SYSTEMS INTERCONNECTION)**
ISO (International Organization for Standardization). Разработана организаций ISO в 1984. Эталонная (теоретическая) модель. **Уровней:** 7. Инкапсуляция данных (добавление заголовков/хвостов на каждом уровне)

|#|Уровень|Единица данных (PDU)|Функции|Примеры протоколов|
|---|---|---|---|---|
|**7**|**Прикладной**|Сообщение|Взаимодействие приложений с сетью|HTTP, FTP, SMTP, DNS, SSH|
|**6**|**Представления**|Сообщение|Форматирование, шифрование, сжатие|SSL/TLS, JPEG, ASCII, Unicode|
|**5**|**Сеанса**|Сообщение|Управление сеансами связи|NetBIOS, RPC, PPTP|
|**4**|**Транспортный**|Сегмент/Дейтаграмма|Надёжная передача end-to-end|TCP, UDP|
|**3**|**Сетевой**|Пакет|Маршрутизация, адресация IP|IP, ICMP, OSPF, BGP|
|**2**|**Канальный**|Кадр (Frame)|Передача между соседними узлами|Ethernet, Wi-Fi, PPP|
|**1**|**Физический**|Бит|Передача битов по физической среде|RJ-45, Fiber, Radio|
